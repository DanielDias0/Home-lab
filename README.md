# Home-lab

Este repositório contém projetos, testes e documentações do meu ambiente de Home Lab,
voltado para aprendizado prático em infraestrutura, redes e automação.

## Objetivos
- Estudar Docker e containers
- Criar serviços de rede (VPN, DNS, etc.)
- Automatizar tarefas com scripts
- Documentar soluções reais

## Estrutura do repositório
- `docker/` – Serviços em containers
- `scripts/` – Scripts Linux e Windows
- `network/` – Configurações e topologia de rede
- `docs/` – Documentação técnica

## Aviso
Projeto para fins educacionais.
